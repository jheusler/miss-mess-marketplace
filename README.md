## What This App Does

Miss Mess Marketplace is built for estate sale enthusiasts, thrifters, resellers, and antique hunters. It currently has two core features:

### 🏷 Sale Discovery Feed
- Browse nearby estate sales, garage sales, and thrift stores in the St. Louis area
- Filter by sale type (Estate, Garage, Thrift)
- Search by keyword, city, or item tag
- View sale details including address, dates, description, and featured items
- Get negotiation tips specific to each sale type
- Open directions in Apple Maps or Waze
- Heart/favorite any sale

### 📷 AI Item Identifier
- Upload up to 4 photos of any item you find at a sale
- AI analyzes the photos using eBay sold comps and returns:
  - Item name and era
  - Original retail price estimate
  - Suggested purchase price (what to pay)
  - Potential resale value (based on eBay sold listings)
  - 2–3 interesting facts about the piece
- Save items to organized folders for later reference

### 🗂 My Finds
- Browse all your saved items in a visual grid
- Filter by folder (Furniture, Jewelry, Art & Decor, Vintage, Electronic
